## discord-bot

克里  
一位OC人給自家OC做的discord-bot

### 功能

- 隨機
- 因數
- 單抽 / 十連
- 神界小遊戲

### 啟動 

使用連結加入

https://discord.com/oauth2/authorize?client_id=1286350395278360629

或是  

clone這個專案  
去discord deveopers 生一個 token  
新增檔案`config.json`  

```
{
    "token": "your token goes here"
}
```

在終端機輸入  

```
npm install
node .
```
